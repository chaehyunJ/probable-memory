vscode 초기 설정

Ctrl + , : 설정

설정 - 검색 - zoom	: 확대/축소
설정 - 검색 - theme	: 색상 테마
설정 - 검색 - asso	: 확장자 연결

메뉴 - 파일 - 자동 저장

extensions (좌측 아이콘 맨 아래) - live server 검색 - 설치
확장 설정 아래쪽 127.0.0.1 값을 자신의 IP주소로 변경
이후 파일 편집중 우측 하단의 Go Live 를 클릭하면 브라우저에서 실행
(방화벽 허용, 연결 프로그램: Chrome, Edge...)
