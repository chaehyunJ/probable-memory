도서 관리 사이트 샘플을 작성하세요

도서 목록은 ArrayList형태로 관리되며, application 내장 객체에 저장되어 있어야 합니다
기능은 등록, 목록, 검색, 수정, 삭제 입니다	(CRUD)
도서 객체의 구성은 아래와 같습니다

- 도서 이름, 저자, 출판일자, 출판사, 가격, 소개

각 메뉴의 링크는 nav태그를 활용한 상단 메뉴로 구성하며
메뉴 페이지는 include 기법을 활용하여 공통 포함시킬수 있도록 처리합니다 <-
필요하다면, 더미데이터를 준비할 수 있습니다

BookDTO[] arr = { ... };
ArrayList<BookDTO> list = new ArrayList(Arrays.asList(arr));

목록은 table 를 활용하여 나타냅니다
공통으로 상단에 페이지 로고를 작성하여, 로고를 클릭하면 항상 대문페이지로 이동할수 있어야 합니다
